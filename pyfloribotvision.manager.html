ContextManager Module¶

class pyfloribotvision.manager.ContextManager.ContextManager(contextName=None)[source]¶

Bases: object

Prepare Application-Context and give the ability to access the Context trough normal initialisation for a given Pseudo-Namespace. The ContextManager provides the Plugins-, Configuration- and DataLink-Controller as well as the ProcessManager.
